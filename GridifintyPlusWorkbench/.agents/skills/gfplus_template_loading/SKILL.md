# GFPlus Template Loading

All shapes in this workbench are generated by loading parametric FreeCAD template files,
writing parameter values into a `Spreadsheet` object, recomputing, extracting the resulting
shape, and immediately closing the template document.

---

## Standard Pattern

```python
import FreeCAD

def load_from_template(template_path, params: dict, feature_name: str):
    """Open template, set spreadsheet params, recompute, return shape copy."""
    try:
        doc = FreeCAD.open(template_path)
    except Exception as e:
        FreeCAD.Console.PrintError(f"[GFPlus] load_from_template: failed to open {template_path}: {e}\n")
        return None

    spreadsheet = doc.getObject("Spreadsheet")
    if spreadsheet:
        for key, value in params.items():
            spreadsheet.set(key, str(value))
        doc.recompute()
    else:
        FreeCAD.Console.PrintWarning(f"[GFPlus] load_from_template: no Spreadsheet in {template_path}\n")

    feature = doc.getObject(feature_name)
    if feature is None:
        FreeCAD.Console.PrintError(f"[GFPlus] load_from_template: no feature '{feature_name}' in {template_path}\n")
        FreeCAD.closeDocument(doc.Name)
        return None

    shape = feature.Shape.copy()   # copy before closing the document
    FreeCAD.closeDocument(doc.Name)
    return shape
```

## Critical Rules

1. **Always `.copy()` the shape** before closing the document â€” the original shape lives
   inside the now-closed document and will be invalid without the copy.
2. **Always call `FreeCAD.closeDocument(doc.Name)`** after extraction â€” leaving template
   documents open pollutes the document list and wastes memory.
3. **Never reuse the same document object** after closing it.
4. **Call `doc.recompute()` after setting spreadsheet values** â€” without this the output
   feature still has the old shape.

---

## Gotchas

- `FreeCAD.open()` opens the file as a new FreeCAD document AND adds it to the document
  list. If you open the same template twice without closing, FreeCAD appends `_1`, `_2`
  to the document name. Always close after use.
- `doc.getObject("FeatureName")` returns `None` if the name doesn't match exactly â€” it is
  case-sensitive and underscore-sensitive (`Pad005` â‰  `pad005`).
- After `recompute()`, check that the feature's `Shape.isNull()` is `False` before copying.
